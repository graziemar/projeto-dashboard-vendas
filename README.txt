# Dashboard de Vendas e Performance Comercial

## Objetivo

Este projeto foi desenvolvido com o objetivo de analisar indicadores comerciais e identificar padrões de vendas, faturamento, performance de vendedores e distribuição regional, utilizando Python e Power BI para construção de análises e dashboards executivos.

---

## Ferramentas Utilizadas

* Python
* Pandas
* Google Colab
* Power BI
* GitHub

---

## Indicadores Desenvolvidos

* Faturamento Total
* Total de Vendas
* Ticket Médio
* Total de Clientes
* Produtos Vendidos

---

## Análises Realizadas

* Evolução de vendas ao longo do tempo
* Faturamento por categoria
* Performance por vendedor
* Distribuição regional de vendas
* Acompanhamento de metas comerciais

---

## Principais Insights

* A categoria Informática apresentou maior faturamento no período analisado.
* A região Sudeste concentrou o maior volume de vendas.
* Os vendedores Ana e Julia apresentaram maior performance comercial.
* O faturamento demonstrou crescimento consistente ao longo dos meses.
